# fusion360gallery-to-build123d
[![Pixi Badge](https://img.shields.io/endpoint?url=https://raw.githubusercontent.com/prefix-dev/pixi/main/assets/badge/v0.json)](https://pixi.sh)

CAD operation sequence importer from the Fusion360 gallery to the Build123D python library

## Description
The goal of this project is to be able to reconstruct the [Fusion360 Gallery reconstruction dataset](https://github.com/AutodeskAILab/Fusion360GalleryDataset/blob/kddw/extrude-curves/docs/reconstruction.md) using the [Build123d](https://build123d.readthedocs.io/en/latest/) python library. 

This project is powered by [pixi](https://pixi.sh/latest/installation/#__tabbed_1_1).

## Installation
```shell
pixi install
```

## Tests
```shell
pixi run test
```
